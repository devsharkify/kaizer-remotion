⚠ CORS BLOCKED — SARVAM API DOES NOT ALLOW DIRECT BROWSER CALLS
The Sarvam API requires server-side calls. Two options to fix:
OPTION 1 — Deploy 1-line proxy on Railway (recommended, free)
// proxy.js — deploy this on Railway const express = require('express'); const { createProxyMiddleware } = require('http-proxy-middleware'); const app = express(); app.use('/sarvam', createProxyMiddleware({ target: 'https://api.sarvam.ai', changeOrigin: true, pathRewrite: { '^/sarvam': '' }, on: { proxyReq: (req) => { req.setHeader('api-subscription-key', process.env.SARVAM_KEY); } } })); app.listen(process.env.PORT || 3000);
Then set PROXY_URL in the field below to your Railway URL (e.g. https://my-proxy.railway.app/sarvam)
